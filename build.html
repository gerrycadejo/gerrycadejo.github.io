Instructions:

1. As a group, pick three pillars for your solution.

Stone Temple Pillars
Example:

Pillars: Easy - Beautiful - fun
R/ It will be easy by implementing a template.
R/ It will be beautiful by having animations that show only one part at a time.
R/ It will be fun by having animated feedback when you complete each stage.

made with ❤ by Q group

JOURNEY
Instructions:

1. The facilitator will read the questions.
2. The group should agree on one answer.
3. The facilitator will type the answers on the text area.
JOURNEY
Example:
Idea: scheduling events.

1. How would the user access your solution?
R/ The user will access the solution by clicking on the calendar icon that is in their profile page.
2. What is the main interaction of your solution?
R/ The user will select a date, a location and a time for the event.
3. What is the user's goal?
R/ With this the user will be able to keep track of when, what and where things are happening.
4. How would they know that they achieved their goal?
R/ By the end of the interaction the user should get a confirmation email.
